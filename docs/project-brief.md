# Project Brief

## Project Name

RentalBills

## One-line Idea

สร้างบิลค่าน้ำ ค่าไฟแล้วส่งผ่าน line ที่กำหนดไว้อัตโนมัติ โดยมี SDK ของ https://github.com/qorstack/qorstack-report ที่เป็นกาสร้างเอกสารแบบใส่ตัวแปรให้ใช้ได้ พร้อม dashboard สำหรับคนที่โอนแล้ว และบันทึกบิลประวัติค่าน้ำค่าไฟของแต่เดือนได้

## Problem

ต้องสร้างบิลใหม่่ทุกเดือนทั้งๆที่ form เดิมทุกเดือน และต้องส่งผ่าน line ทีละคน บางครั้งก็ส่งให้ผิดคน และส่งไม่ทันรอบเดือนบ้างเนื่องจากทำไม่ทัน

## Target Users

ผู้เช่าห้องปะมาณ 15 ห้อง

## Main Goal

อยากได้ระบบที่จัดการเอกสารแบบตัวแปร และส่งบิิล PDF ไปทั้งค่าน้ำและค่าไฟผ่าน line ที่สามารถกำหนดบุคคลได้้ และสามารถเช็คยอดเงินได้ว่าใครโอนแล้ว ใครยังไม่โอน และส่ง statement ไปใน line สำหรับคนที่โอนแล้ว

## Core Features

1. ระบบสร้างบิลค่าน้ำค่าไฟ
2. ระบบส่งบิลผ่าน line แบบกำหนดบุคคลได้
3. บันทึกบิิลแต่ละเดือนเก็บไว้โหลดซ้ำได้
4. QR code สำหรับส่งธนาคารที่ต้องโอนให้ลูกค้า และข้อความตอบกลับว่าโอนแล้ว

## Nice-to-have Features

1. Dashboard สำหรับคนที่โอนแล้วและยังไม่โอน
2. ระบบร้องเรียนเมื่อเกิดปัญหา โดยลูกค้าสามาถแจ้งผ่าน line
3. ระบบผู้ใช้และสิทธิ์
